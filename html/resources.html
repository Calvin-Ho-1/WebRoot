Web Resources

W3Schools | CodeAcademy | Khan Academy | GeeksforGeeks | StackOverFlow | Main

  1. W3Schools
    W3Schools is one of the largest web developer site. It's very well known from many users for its text tutorials which includes HTML, CSS, Javascript and much more.
    I have used W3Schools before and it shows basic examples for the user to understand what it should look like and what each specific tag does. It's very helpful and easy to understand for beginners.
    Rating: 5/5

  2. CodeAcademy
    CodeAcademy is a platform that offers free coding classes in many different programming languages.
    I have heard of CodeAcademy before, but I haven't really tried it out until I started a course where it introduced me to it (ISIT 320). It introduces you to the basics and allows the user to perform the code. This is good for beginners wanting to know how to code or a refresher for those who have coded before. This allows the user to know the fundamentals while also teaching them how to perform them.
    Rating: 5/5

  3. Khan Academy
    Khan Academy is a non-profit organization that offers a variety of lessons in videos that helps educate students.
    Khan Academy gives video lessons on the basics of programming. However, there's not too much on specific programming languages that they'll teach. Khan Academy is a great website, but it doesn't offer too much programming lessons compared to others.
    Rating: 2.5/5

  4. GeeksForGeeks
    GeeksforGeeks is an organization that offers free tutorials and provides code examples that help students and those who are more experienced in programming.
    I think that this website can provide specific ideas and requirements for what the user might wanted. It can also provide those who are looking for jobs and prepare for interview questions. It has many programming languages that can provide help for people. The main downside that I have for this website is its searchibility for programming languages.
    Rating: 4/5

  5. StackOverflow
    StackOverflow is a Q&A Website for programmers. It allows experienced programmers to ask other programmers around the world.
    StackOverflow is a good website for those who are already well experienced into programming. It's good for searching for some specific code that you might need. However, I think that it won't offer as great of benefits for those who are still new to programming. This is a Q&A website so it's highly recommended for those who are new to use other websites.
    Rating: 2/5 (for beginners)
